# CRM steg 10 fixed

Den här versionen innehåller ett komplett Next.js-projekt som går att deploya på Vercel.

## Innehåll
- startsida i `app/page.tsx`
- activity API
- Outlook token storage endpoint
- Telavox mock call endpoint
- enkel UI-översikt

## Start lokalt
```bash
npm install
npm run dev
```
